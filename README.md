# jTerm

macOS용 SSH/SFTP 터미널 애플리케이션

## 주요 기능

- 🖥️ **로컬 터미널** - 기본 터미널 기능
- 🔐 **SSH 연결** - 비밀번호 및 SSH 키 인증 지원
- 📁 **SFTP 파일 관리** - 드래그 앤 드롭 파일 전송
- 🎨 **다양한 테마** - Dark, Light, Dracula, Nord 등 9가지 테마
- 🌍 **다국어 지원** - 한국어, 영어, 일본어
- 🔄 **자동 업데이트** - 새 버전 자동 알림
- 🔑 **Keychain 통합** - SSH 키 및 비밀번호 안전 저장
- 🆔 **Touch ID 지원** - 생체 인증으로 앱 보호
