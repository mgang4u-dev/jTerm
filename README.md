# jTerm

macOS용 터미널 애플리케이션

<br clear="left">

## 기능

- 로컬 터미널
- SSH 연결 관리
- SFTP 파일 전송
- 다중 탭/창 지원
- 다크/라이트 테마
- 자동 업데이트

## 설치

[Releases](https://github.com/mgang4u-dev/jTerm/releases)에서 최신 DMG 파일을 다운로드하세요.

## 시스템 요구사항

- macOS 10.15 이상
- Apple Silicon (M1/M2/M3)
